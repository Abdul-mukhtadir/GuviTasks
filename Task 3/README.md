# Memory Card Game

A simple and responsive Memory Card Game built using HTML, CSS, and JavaScript.

## Features
- Card flipping animation
- Random card shuffling
- Match checking logic
- Restart functionality
- Fully responsive (desktop & mobile)
- Clean and easy-to-understand code

## Setup Instructions
1. Download or clone the project.
2. Make sure all files are in the same folder:
   - index.html
   - style.css
   - script.js
3. Open `index.html` in any modern web browser.

## How to Play
1. Click on any card to flip it.
2. Click on another card to find its matching pair.
3. If both cards match, they stay face-up.
4. If they do not match, they flip back after a short delay.
5. Continue until all pairs are matched.
6. Click the **Restart** button to reshuffle and play again.

## Technologies Used
- HTML for structure
- CSS for styling and responsiveness
- JavaScript for game logic and DOM manipulation
