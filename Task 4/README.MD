# Income & Expense Tracker

A simple and responsive web application to manage income and expenses using HTML, CSS, and JavaScript.

## Features
- Add, edit, and delete income/expense entries
- Filter transactions (All / Income / Expense)
- Live calculation of total income, total expenses, and balance
- LocalStorage support (data persists after refresh)
- Fully responsive design
- Clean CRUD implementation

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)


## CRUD Operations
- **Create:** Add new transactions
- **Read:** View transaction list
- **Update:** Edit existing entries
- **Delete:** Remove entries
